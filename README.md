# ALADIN_Cryptography

I created this project during my studies.
It's purpose is to generate exercises for students in different difficulties and with solution aids.

Consisting out of RSA.

Used technologies:
* Vite
* React
* Bootstrap
* Typescript

## Installation

```
cd src
npm install
cd frontend
npm install
```

## Testing

```
cd src/frontend
npm run dev
```

## Build

```
cd src/frontend
npm run build
```
